Tackling Multiple Visual Artifacts: Blind Image Restoration using Conditional Adversarial Networks (ongoing)
V Jeya Maria Jose, M Anand, A Ashwin Natraj, K Subramanian, S Deivalakshmi

Restoring images that are degraded by multiple visual artifacts like noise, blurness and other environmental visual artifacts like shadow, snow, rain and haze is a challenging task. In this work, use of conditional adversarial networks for this task has been explored. ISTD dataset augmented with different types of noise and blurness has been used to train the network.
